Reflectivity / Line Following Sensor VUPN6320

This module measures the reflectivity of a surface with an infrared emitter/detector pair. The output goes to high whenever the reflectivity exceeds the threshold value, which can be adjusted with the onboard potentiometer.

The reflectivity sensor module is built around the venerable LM393 comparator IC. Only 3 wires are needed for operation, V+ (5VDC), GND, and a digital pin. All pins are standard 0.1" spaced
