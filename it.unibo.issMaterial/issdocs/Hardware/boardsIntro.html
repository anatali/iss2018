Windows 7 as hotspot WIFI

Open a DOS window as administrator and insert the following commands: 

    netsh
    wlan set hostednetwork mode=allow (or disallow)
    wlan set hostednetwork ssid=natspot
    wlan set hostednetwork key=123456789 keyUsage=persistent
    wlan start hostednetwork
Now you can check for the presence in Microsoft Device Manager of a Virtual WiFi Miniport Adaptor.

By opening Centro connessioni di rete e condivisione you should see a second adapter of wireless network.

Ora con tasto destro del mouse sulla scheda di rete "reale" (quella che il computer utilizza per accedere ad Internet) scegliere la voce Proprietà. Nella scheda Condivisione, bisognerà spuntare la casella Consenti ad altri utenti in rete di collegarsi tramite la connessione Internet di questo computer, disattivare Consenti ad altri utenti in rete di controllare o disabilitare la connessione Internet condivisa quindi selezionare - dal menù a tendina Connessione rete domestica - la rete utilizzata dal computer Windows 7 per connettersi ad Internet:

To close and stop 

    netsh
    wlan stop hostednetwork
    wlan set hostednetwork mode=disallow
Disinstallare the second adapter of wireless network per consentire la connessione con Raspberry su IP 192.168.137.2.

USB Tethering on Raspberry

    sudo nano /etc/network/interfaces
    auto eth0
    iface eth0 inet dhcp
    iface usb0 inet dhcp   
    sudo ifup usb0   
    sudo reboot
