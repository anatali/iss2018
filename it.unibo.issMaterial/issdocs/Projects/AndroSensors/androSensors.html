Android sensors: Overview

  1. The application CtxButton (DevButtonAndroid) shows how a Button can be modelled and implemented as a QActor that sends a dispatch to another, remote QActor or as a event-emitter.
  2. The application CtxAndroEvent (MainActivity) shows how an Android sensor (accelerator) can be modelled and implemented as a event emitter
    3. by introducing a proper observer.
  3. The application CtxAndroStream shows how Android sensors (accelerator, proximity, ...) can be modelled and implemented as event streanms that can be merged and then 'trnasformed' into events by a proper subscriber (SensAndroSrcObserverJson).
