RobotActor

The RobotActor is a that allows us to interact with a BaseRobot. It provides the operations (see IRobotActor): 

long execute(String command, int speed, int angle, int millisec, String  evId)
long execute(String command, String speed, String millisec, String angle, String evId)
long executeTheCommand(IBaseRobotCommand command, int millisec,String  evId)
Moreover it allws:
  • to receive commands by interacting via HTTP (on port 8080) with a browser
  • to receive commands as messages sent by other QActors or other software components
  • to use a stream of sensor-data coming from a Android smart device
  • to use streams of sensor-data coming from the onborad senors
  • to raise or perceive events
  • to emit sounds or to capture photo or video
