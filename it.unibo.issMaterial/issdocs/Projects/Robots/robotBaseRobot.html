BaseRobot

The BaseRobot is a POJO that can be called to execute moves. 

it.unibo.iot.baseRobot.configurator.Configurator configurator = Configurator.getInstance();
it.unibo.iot.executors.baseRobot.IBaseRobot robot = configurator.getBaseRobot();

It could also provide a set of on-boord observable POJO sensors: 

 it.unibo.iot.sensor.ISensor sensor : configurator.getSensors()
 if( sensor instanceof DistanceSensor){
    DistanceSensor sensorLine = (DistanceSensor) sensor;
    ISensorObserver obs = new DistanceSensorObserver(outView);
    sensorLine.addObserver(  obs  ) ;
 }
