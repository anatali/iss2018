LOW-LEVEL

The following project is a very simple introduction to the usage of RaspberryPi GPIO by means of bash commands, gpio library and Python.

Project it.unibo.raspIntro

Shell (see Raspberry Pi GPIO via the Shell)
Package Entry Goal
it.unibo.bls.bash led25OnOff.sh manage a Led connected on the GPIO pin by reading/writing a (virtual) file associated with that pin
it.unibo.bls.bash buttonOn24Click.sh manage a Button connected on the GPIO pin by reading/writing a (virtual) file associated with that pin
it.unibo.bls.bash buttonLed.sh the BLS system implemented by reading/writing some (virtual) file associated with the device pins

gpio library (see Command-line utility gpio)
Package Entry Goal
it.unibo.bls.bash.gpio led25Gpio.sh manage in bash a LED connected on a GPIO pin 25 (Wpi 6) by using the GPIO library
it.unibo.bls.bash.gpio button24Gpio.sh manage in bash a Button connected on a GPIO pin 24 (Wpi 5) by using the GPIO library
it.unibo.bls.gpio BLSGpio.java manage in Java a LED connected on a GPIO pin 25 (Wpi 6) by using the GPIO library

python (see gpio-pins-and-python)
Package Entry Goal
it.unibo.bls.python ledPython25.py manage a Led connected on the GPIO pin 25 using the python language.
it.unibo.bls.python buttonPython24.py manage a Button connected on the GPIO pin 24 using polling.
it.unibo.bls.python buttonPython24Edge.py manage a Button connected on the GPIO pin 24 using wait_for_edge.
it.unibo.bls.python buttonPython24EventDetected.py manage a Button connected on the GPIO pin 24 using event_detected.
