Workflow

1) Define a Observable.OnSubscribe

The interface Observable.OnSubscribe is related to any object that can be used as a stream source of data of type T.

The method call of Observable.OnSubscribe (inherited from interface rx.functions.Action1, in its turn inherited by the interface rx.Observable.OnSubscribe) is executed each time a Subscriber subscribes to our source.

Example: OnSubscribeSimpleGen

2) Create a stream source rx.Observable

The stream source is created by Observable.create that returns an rx.Observable that will execute the specified function when a Subscriber subscribes to it.

Example: ObservableSimpleGenMain

3) Add some subscriber to the stream

The application work can be done by a subscriber added to the stream source

Example:
ObservableSimpleGenMain
